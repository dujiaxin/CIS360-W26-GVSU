By the end of this course, students will be able to:

- Explain core concepts in information management and information science
- Describe and compare relational, document, graph, and vector databases
- Interpret and model information using simple schemas and diagrams
- Understand how modern AI systems retrieve and use structured data
- Evaluate ethical and responsible use of information systems
